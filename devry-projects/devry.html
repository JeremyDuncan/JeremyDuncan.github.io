These are my projects from DeVry

PHYS204 Final Project:

The Application of IoT Technology and Physics

This is the final project for my DeVry class PHYS204. In this project I will be demonstrating fundamental concepts of the governing laws of physics by applying technologies from the growing field of Internet of Things (IoT).

NETW211 Final Project:

Cloud Computing Fundamentals

This is the final project for my DeVry class NETW211. In this project I walk you through fundamental cloud computing concepts while using Amazon Web Services (AWS).

CEIS101 Final Project:

IoT: Internet of Things

This is the final project for my DeVry class CEIS101. In this project I walk you through fundamental IoT concepts while using Arduino components to make an automated home security system.

CEIS150 Final Project:

Object Oriented Programming

This is the final project for my DeVry class CEIS150. In this project I walk you through a stock tracking program I created using Python.

SEC285 Final Project:

Information Security

This is the final project for my DeVry class SEC285. In this project I walk you through information security fundamentals.

About

I have been going to DeVry University full-time working towards a Bachelor's of Science in Computer Information Systems with a focus in Software Programming.
